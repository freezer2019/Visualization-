Summary: Latitude Vs. X

Dashboard displaying latitude images produced from a random selection of 500 unique cities.

Weather checks were performed on a variety of cities using a series of successive API calls.

An analysis of the data was conducted to view the changes of weather as you move closer to the equator.

Matplotlib was used to produce the scatter plots of weather vs. latitude. The following weather conditions were taken into consideration:

  • Max Temperature

  • Humidity

  • Cloudiness

  • Wind Speed

This dashboard provides the source data and visualizations generated, as well as a written analysis of any observable trends or correlations
